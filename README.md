# Sistema de Academia de Ginástica Artística "Movendo Estrelas"

Este repositório contém os scripts SQL da Experiência Prática IV. 

## Objetivo
Implementar e manipular dados em um banco de dados relacional utilizando comandos SQL (DML), aplicando conceitos de tipos de dados, integridade referencial e versionamento de código.

## Estrutura
- `create_tables.sql` → Criação das tabelas principais com PK, FK e restrições.
- `insert.sql` → Comandos de INSERT para povoar as tabelas principais.
- `select.sql` → Consultas SELECT (JOIN, WHERE, ORDER BY).
- `update.sql` → Comandos UPDATE para modificar dados.
- `delete.sql` → Comandos DELETE para remover registros.

## Como executar
1. Crie um banco de dados no PostgreSQL usando PGAdmin (ex.: `UDF`).
2. Execute `create_tables.sql` para criar as tabelas.
3. Execute `insert.sql` para inserir os dados iniciais.
4. Teste as consultas com `select.sql`.
5. Aplique modificações com `update.sql`.
6. Remova registros com `delete.sql`.

## Requisitos atendidos
- Inserção de dados (INSERT) em todas as tabelas principais.
- Consultas (SELECT) com JOIN, WHERE e ORDER BY.
- Atualizações (UPDATE) em pelo menos três registros.
- Exclusões (DELETE) em pelo menos três registros.
- Scripts organizados e versionados em repositório público.

## Ferramentas utilizadas
- PostgreSQL + PGAdmin
- GitHub para versionamento e entrega
